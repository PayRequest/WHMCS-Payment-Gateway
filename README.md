# PayRequest WHMCS Payment Gateway
WHMCS Payment Gateway Module for PayRequest

Payment Gateway for connecting WHMCS with the PayRequest, and subsequently catching the PayRequest webhooks to mark WHMCS invoices as "paid" on successful payment.

# Installation
1. Upload the payrequest.php file and "callback" folder to your modules/gateways/ folder of your WHMCS installation.


## Usage and support
This Module is supported by PayRequest, we offer technical support and updates, please open an issue if you need any help.

For more information, please refer to the online documentation for WHMCS and Paddle:
https://developers.whmcs.com/payment-gateways
https://paddle.com/docs/api-custom-checkout/
